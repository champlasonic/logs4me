# Welcome

This is How to write.

## How to Use Markdown

You can write posts in Markdown format on this blog.

### Text Formatting

- **Bold**
- _Italic_
- ~~Strikethrough~~

### Lists

1. Ordered list item one
2. Ordered list item two
3. Ordered list item three

- Unordered list item
- Another item
- Yet another item

### Code

Inline code: `console.log('Hello, World!')`

Code block:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

### Links

[Link to GitHub](https://github.com)

### Images

To add an image, use this syntax:

```markdown
![Alt text](image-url)
```

## How to Create a New Post

1. Navigate to the `_posts/` folder in your GitHub repository
2. Click "Add file" → "Create new file"
3. Enter filename in `YYYY-MM-DD-title.md` format
4. Write YAML frontmatter and content
5. Click "Commit changes" to save

Your post will be automatically published on GitHub Pages in a few minutes.

Happy blogging!
